🚀 البدء السريع (Quick Start)
═══════════════════════════════════════════════════════════════════════════════

▶️ لنظام Linux/Mac:

   1. فك الضغط:
      unzip subtitle-translator-pro.zip
      cd subtitle-translator-pro

   2. تثبيت المتطلبات:
      ./install.sh

   3. بدء التشغيل:
      ./start.sh

   4. فتح المتصفح:
      http://localhost:5000


▶️ لنظام Windows:

   1. فك الضغط:
      استخرج الملفات من subtitle-translator-pro.zip

   2. تثبيت المتطلبات:
      انقر نقراً مزدوجاً على: install.bat

   3. بدء التشغيل:
      انقر نقراً مزدوجاً على: start.bat

   4. فتح المتصفح:
      http://localhost:5000


⚙️ إعداد مفاتيح API (اختياري)
═══════════════════════════════════════════════════════════════════════════════

لاستخدام خدمات الترجمة المدفوعة (Gemini، DeepL):

1. انسخ ملف .env.example إلى .env:
   cp .env.example .env

2. أضف مفاتيح API الخاصة بك:
   GEMINI_API_KEY=your_key_here
   DEEPL_API_KEY=your_key_here

3. احفظ الملف وأعد تشغيل الخادم

ملاحظة: Google Translate يعمل مجاناً بدون API key!

🐛 حل المشاكل الشائعة
═══════════════════════════════════════════════════════════════════════════════

❌ خطأ: "Python not found"
   ✅ تأكد من تثبيت Python 3.8+
   ✅ أضف Python إلى PATH

❌ خطأ: "pip install failed"
   ✅ استخدم: python -m pip install -r requirements.txt
   ✅ جرّب مع sudo/admin (Linux/Mac)

❌ خطأ: "Connection refused"
   ✅ تحقق من اتصال الإنترنت
   ✅ تأكد من المنفذ 5000 غير مستخدم

❌ خطأ: "Translation service unavailable"
   ✅ تحقق من مفتاح API في .env
   ✅ تحقق من اتصال الإنترنت

❌ الترجمة بطيئة جداً
   ✅ استخدم Google Translate للسرعة
   ✅ قلل عدد الملفات
   ✅ عطّل حفظ السياق
